# MADock
任意の映像と音声を結合するツール

前提条件：ffmpegの導入  
（入手先→ https://ffmpeg.org）

## 使用方法
- batファイルからスクリプトを起動する  
（初回起動時はffmpeg.exe を指定する案内が表示されます）
- 結合したい映像ファイルと音声ファイルをドラッグ&ドロップする
- ファイルが揃ったら実行ボタンを押すと処理が始まります　　
（初期出力先はスクリプトファイル保存場所）

## 対応形式
### 入力ファイル
- 動画: mp4
- 音声: wav, FLAC, ALAC

### 出力ファイル
- mp4: 映像は無劣化、音声はAACに変換
- mkv: 映像・音声ともに無劣化
